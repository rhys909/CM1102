Functionality of Python

Introduction

This page will talk about some of the features and functionalities of the Python programming language. It will also provide some insight into the many uses of Python and what makes it one of the top 3 languages committed to GitHub.

The Python language is a multi paradigm language, meaning that the programmer can choose to use the programming style which suits the needs of that individual project. Python supports both structured and object-oriented programming fully. However Python can support other paradigms through the use of extensions.

Standard Library

The standard Python 3.7.1 install comes with a wealth of built-in functions and facilities. These include a library of Pythons built in functions.

Built in Functions

Python has a large number of built in functions these include:

  • abs()
  • all()
  • float()
  • int()
  • str()
  • type()
  • list()
  • range()
  • help()
  • slice()
  • repr()
  • __import__()
